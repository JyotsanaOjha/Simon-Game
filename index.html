SIMON GAME

How To Play? HOW TO PLAY?
  • Press any key to start.
  • Simon will give the first signal. (One of the Colors will blink, accompanied by a tone.)
  • Repeat that signal by clicking the same Color.
  • Simon will keep track of the first signal and add one more.
  • Repeat those two signals in the same order.
  • Simon will keep track of the past sequence and add one more signal for each subsequent level.
  • Continue to repeat Simon for as long as you can!
VICTORY:

If you fail to complete the sequence in the correct order, the Buzzer will sound, and the game will end.
